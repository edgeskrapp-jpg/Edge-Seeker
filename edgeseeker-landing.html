HOW IT WORKS

01

PROBABILITY MODEL

Edge Seeker runs a multi-layer Poisson model across every MLB game daily. It calculates true win probabilities using team run averages, park factors, pitcher quality, and Elo ratings — then compares against book implied odds to find discrepancies.

02

AI SYNTHESIS

The EDGESEEKER agent powered by Claude Opus synthesizes Poisson outputs with FanGraphs FIP data, Baseball Savant statcast metrics, weather conditions, and injury reports. When multiple signals align — it acts. When they conflict — it passes.
